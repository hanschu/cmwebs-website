導入流程

房東先建檔，房客再報到，之後交給系統追蹤

適合已經有舊房客的房東，也能處理新房客線上簽約。資料量大時可搭配批次上傳表格。

一、房東端開通流程

  1. 加入房東：房東先加入 CMWebs 官方 LINE，完成帳號綁定。
  2. 房東建檔：建立房屋、房間、租金及現有房客資料。房客數量多時，可使用大量上傳表格。
  3. 開通系統：確認資料完整後，開通租管服務。
  4. 通知房客報到：請房客進入系統，透過「房客報到」綁定自己的 LINE 帳號。

二、每月收租流程

  1. 每月抄表：每月 4 號前輸入電表度數，或批次上傳多房間資料。
  2. 自動發帳：系統依租金、電費與設定項目計算帳單，透過 LINE 發給房客。
  3. 房客匯款：房客沿用原本銀行匯款方式，不需另外申請第三方金流。
  4. 回報後五碼：房客回覆匯款帳號後五碼，系統完成銷帳流程並通知房東。
  5. 自動催繳：尚未繳租的房客，由系統持續追蹤並依設定發送提醒。

三、租約到期與退租

合約到期前，系統通知房客確認是否續約。同意續約時，系統可發送新合約供房客簽署並通知房東；不同意續約時，系統提醒房東安排退租點交與相關結算。

四、新房客線上簽約

房東可發起新租約，讓新房客線上填寫資料並簽署。簽約後再完成房客報到，後續帳單與租約追蹤即可納入系統。

實際流程與可用功能仍以正式開通帳號的設定為準。涉及租賃契約、通知期限及催繳程序時，請依契約與現行法令處理。
